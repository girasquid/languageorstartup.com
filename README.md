## Language, or startup?

### Generating the data

Startups
```bash
ruby startups.rb
```
Paste the output into `STARTUPS` in index.html.

Languages
```bash
ruby languages.rb
```
Paste the output into `LANGUAGES` in index.html.

### How did you make the sweet favicon?!

```bash
brew install imagemagick
convert -size 1x1 xc:#39CCCC favicon.png
```
